# Contact Manager Web Application

## Overview
A simple yet powerful contact management system built to help users organize their personal and professional connections efficiently.

## Features
- Add, edit, and delete contacts
- Store essential contact information (name, email, phone, etc.)
- User-friendly interface
- Responsive design for mobile and desktop use
- Search functionality
- Contact categorization

## Technical Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JWT

## Purpose
Created this project to demonstrate modern web development practices while solving a common organizational need. The focus was on delivering a clean, intuitive user experience while maintaining robust data management.

## Learning Outcomes
- Implemented CRUD operations
- Managed state effectively
- Handled user authentication securely
- Applied responsive design principles
- Practiced REST API development

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Configure environment variables
4. Run: `npm start`
